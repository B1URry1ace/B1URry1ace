## 🚀 Ключевые проекты

* 🫀 [**MedVision & QCA Analysis**](https://github.com/B1URry1ace/MedVision-and-QCA)
  Полноценное десктопное приложение (PySide6) для сегментации коронарных ангиограмм. Включает AI-помощника для авторазметки (YOLO), автоматический количественный анализ сосудов (QCA, алгоритм Чжана–Суэня), расчет процента стеноза и генерацию клинических отчетов. Под капотом: строгая модульная архитектура, OpenCV, SQLite.
  
* 🔬 [**Custom Coronary YOLO Segmentation**](https://github.com/B1URry1ace/custom-coronary-yolo-segmentation) 
  Обучение модели сегментации коронарных артерий (YOLO-seg) и её интеграция в CVAT в качестве serverless-функции авторазметки через Nuclio.
  
* ⚙️ [**CVAT Shape-to-Mask API**](https://github.com/B1URry1ace/cvat-shape-to-mask-api)
  Инструмент автоматизации для конвертации векторных полигонов в RLE-маски напрямую в CVAT. Использует официальный CVAT SDK и OpenCV. Позволяет избежать ручной работы с файлами при подготовке датасетов для обучения моделей.
  
* 🤖 [**Schedule Bot**](https://github.com/B1URry1ace/Schedule_bot) 
  Telegram-бот для получения расписания ИКТИБ с поддержкой обработки голосовых заметок.
  
* 💬 [**LM-Studio-Flet-Messenger**](https://github.com/B1URry1ace/LM-Studio-Flet-Messenger) 
  Мобильное кроссплатформенное приложение-мессенджер на Flet для взаимодействия с локальными LLM через LM Studio.
